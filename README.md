# 🤖 Telegram Video Text Remover Bot

بوت تلجرام لإزالة النصوص والكتابة من الفيديوهات باستخدام الذكاء الاصطناعي.

## 🚀 التشغيل على Render

1. انسخ المشروع إلى GitHub
2. أنشئ حساب على [Render](https://render.com)
3. اربط مستودع GitHub
4. أضف متغيرات البيئة في إعدادات Render
5. اختر "Background Worker" كنوع الخدمة

## ⚙️ المتطلبات

- Python 3.9+
- حساب Cloudinary (مجاني)
- بوت تلجرام (من @BotFather)

## 🔧 الإعداد

1. احصل على توكن البوت من @BotFather
2. سجل في Cloudinary واحصل على API Keys
3. أضف المتغيرات في ملف `.env`

## 📦 التثبيت المحلي

```bash
pip install -r requirements.txt
python bot.py
